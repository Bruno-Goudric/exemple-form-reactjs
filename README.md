## Tecnólogias Utilizadas

ReactJs
TypeScript
React-Hook-Forms
Yup
React-router-dom

## Instalação

Você precisará clonar este repositório:

```sh
  $ git clone https://github.com/Bruno-Goudric/exemple-form-reactjs.git
```

Agora você precisa instalar os pacotes:

```sh
  $ yarn install
```

```sh
  $ npm install
```
